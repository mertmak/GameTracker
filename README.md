# 🎮 Oyun Takipçisi (Game Tracker)

Bu proje, oyun koleksiyonunuzu yönetmenizi, oynama sürelerinizi takip etmenizi ve oyun istatistiklerinizi görmenizi sağlayan modern bir **React** uygulamasıdır.

## 🚀 Canlı Demo
Projenin yayındaki haline buradan ulaşabilirsiniz: gametrackermertmak.netlify.app

## 📸 Ekran Görüntüsü

    *Ekran görüntüleri ScreenShots klasörünün içindedir.

## ✨ Özellikler

* **Tam CRUD Operasyonları:** Oyun ekleme, listeleme, sıralama, durum güncelleme ve silme.
* **Akıllı Kontrol:** Aynı isim ve aynı platforma sahip oyunların  eklenmesi engellenir.
* **İstatistik Paneli:** Toplam oyun sayısı, bitenler, oynananlar ve istek listesindekiler için dinamik özet.
* **Gelişmiş Filtreleme:** Oyunları durumlarına ve platformlarına göre anlık filtreleme.
* **Oynama Süresi Takibi:** Her oyun için harcanan saati kaydetme düzenleme ve görüntüleme.
* **Veri Kalıcılığı:** `Local Storage` entegrasyonu sayesinde sayfa yenilense bile verileriniz kaybolmaz.
* **Modern Arayüz:** Tailwind CSS kullanılarak hazırlanmış, karanlık tema destekli şık ve duyarlı (responsive) tasarım.

## 🛠️ Kullanılan Teknolojiler

* **Framework:** [React](https://reactjs.org/) (Vite ile kuruldu)
* **Styling:** [Tailwind CSS](https://tailwindcss.com/)
* **State Management:** React Hooks (`useState`, `useEffect`)
* **Deployment:** [Netlify](https://www.netlify.com/)

## 📂 Proje Yapısı

Yönergeye uygun olarak aşağıdaki klasör yapısı kullanılmıştır:
- `src/Components`: Form, Liste ve İstatistik bileşenleri.
- `src/Pages`: Ana sayfa (Home) tasarımı ve uygulama mantığı.

## ⚙️ Kurulum ve Çalıştırma

Projeyi yerel bilgisayarınızda çalıştırmak için:

### Adım 1: Depoyu Klonlayın
```bash
git clone https://github.com/mertmak/GameTracker.git
```

### Adım 2: Bağımlılıkları Yükleyin
```bash
npm install
```

### Adım 3: Geliştirme Sunucusunu Başlatın
```bash
npm run dev
```
### Adım 4: Üretim İçin Build Yapın
```bash
npm run build
Tarayıcınızda `http://localhost:5173` adresine gidin.
```
